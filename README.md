# Sticker AI Generator

Generador de stickers para WhatsApp usando inteligencia artificial.

## Demo

[https://sticker-ai-generator.vercel.app](https://sticker-ai-generator.vercel.app)

## Funcionalidades

- Genera stickers personalizados con IA
- Descarga en formato WebP compatible con WhatsApp
- Interfaz intuitiva y responsive

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- OpenAI API

## Instalacion

```bash
npm install
```

## Configuracion

Crea un archivo `.env.local` con:

```
OPENAI_API_KEY=tu_api_key
```

## Desarrollo

```bash
npm run dev
```
